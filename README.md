# 🎓 Dr. Babasaheb Ambedkar Technological University (DBATU) - Learning Management System

Welcome to the **DBATU LMS**, a highly scalable, premium-grade Learning Management System completely engineered from scratch using structural HTML5, vanilla JavaScript, and cascading CSS3 paradigms without external framework dependencies.

## 🚀 Project Overview
This repository contains a full-stack Front-End implementation mapped specifically for university-grade educational frameworks. The system features:
- **Responsive Architecture**: Fully responsive across Laptop, Tablet (768px), and Mobile (480px) configurations using native Flexbox/Grid routing.
- **Secure Authentication Mocks**: Fully modeled interactive Login and Registration interfaces utilizing structural DOM evaluation.
- **Interactive Modules**: Complete execution of 12 fully interactive sub-systems, including live Quizzes, Forums, Assignments, and Administrative panels.
- **Thematic Integrity**: Built precisely to DBATU specifications utilizing Royal Navy (`#1a2a4a`) and Academic Gold (`#f5a623`) with integrated typography (`Inter` & `Poppins`).

## 🖥️ Local Execution & Testing

Running the DBATU LMS locally requires virtually zero configurations:

1. **Download/Clone** the repository to your local machine.
2. Navigate directly into the root folder (`Project_Moodle`).
3. Simply double-click on `index.html` to execute the system in your default browser (Chrome, Firefox, Safari recommended).
4. No Node servers, dependencies, or compilers are required.

> **Note:** Because this platform uses standard relative paths `../pages/`, navigating between pages is completely seamless natively offline!

## 🌍 Free Deployment Guide (GitHub Pages & Netlify)

This statically generated frontend system is perfectly suited to be globally hosted for free using leading CDN services.

### Option A: GitHub Pages (Recommended)
1. Log into your GitHub account and create a new repository (e.g., `dbatu-lms`).
2. Upload all the files inside the `Project_Moodle` directory directly into the repository.
3. Once pushed, navigate to **Settings** > **Pages** (on the left sidebar).
4. Under the **"Build and deployment"** source dropdown, select `Deploy from a branch`.
5. For the **Branch**, select `main` (or `master`) and hit **Save**.
6. Wait roughly 1-2 minutes. GitHub will generate a live URL (e.g., `https://yourusername.github.io/dbatu-lms/`) where the system is 100% accessible globally!

### Option B: Netlify
1. Go to [Netlify Core](https://app.netlify.com/drop) and either log in or sign up securely.
2. Locate the "Drag and Drop" deployment area on the Sites dashboard.
3. Simply grab the entire `Project_Moodle` directory from your computer and drop it directly into the Netlify build window.
4. Netlify will instantly compile and publish the site in 5-10 seconds automatically.
5. Click perfectly on the generated URL to access your live system.
6. *(Optional)* Navigate to "Site Settings" to explicitly change the domain name to something clean like `dbatu-lms.netlify.app`.
